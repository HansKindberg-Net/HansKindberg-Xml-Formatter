HansKindberg-Xml-Formatter
==========================
If you can not start debugging the VS-package (HansKindberg.VisualStudio.Extensions.XmlFormatter) project:

1. Right click the VS-package project (HansKindberg.VisualStudio.Extensions.XmlFormatter)
2. Choose Properties
3. Choose the Debug tab
4. In the Start Action area enable Start external program and set it to “C:\Program Files (x86)\Microsoft Visual Studio 11.0\Common7\IDE\devenv.exe” (or whatever Visual Studio version you have)
5. In the Start Options area add “/rootsuffix Exp” (this will start the Experimental instance of Visual Studio)
